# Evidence Capture Guide

This document explains how to regenerate the verification artifacts included in this repository.

## Steps

1. Run

   powershell -ExecutionPolicy Bypass -File scripts/capture_evidence.ps1

2. Save screenshots of each generated artifact using the filenames below.

Required artifacts

- terminal_execution.txt
- replay_evidence.json
- pytest_execution.txt
- coverage_report.txt
- benchmark_output.txt
- project_structure.txt
